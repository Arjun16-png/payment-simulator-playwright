# Payment Transaction Simulator (Playwright)

## 📌 Overview
This project simulates a basic payment transaction system and automates test scenarios using Playwright.

## 🎯 Purpose
To demonstrate SDET capabilities including:
- Test automation
- Understanding of transaction flow
- Handling edge cases in payment systems

## 🔧 Tech Stack
- Playwright
- TypeScript
- HTML (Mock UI)

## 🧪 Test Scenarios
- Successful payment (00)
- Insufficient funds (51)
- Invalid amount
- Input validation

## ▶️ How to Run
```bash
npm install
npx playwright test
